Windows
====================================
Click on the folder and type "cmd"
In the command prompt type "pip install -r requirements.txt"
Then type gassed.py
===================================
Linux/Termux
===================================
You need Python3 Installed.
When it's cloned type "cd GasDoS".
Then type "pip3 install -r requirements.txt".
Then type "python3 gassed.py".
===================================
